\abstract{This report introduces the NPS \LaTeX{} template
  which can be used to produce a master's thesis, a
  Ph.D.\ dissertation, or a technical report. The template can produce
  documents that are unclassified, For Official Use Only (FOUO), or classified.
  Additional information on using \LaTeX\,is also provided.
  This document is a strict update to NPS technical report NPS-CS-11-011 (by
  Simson L.\ Garfinkel and Travis W.\ Axtell), which describes
  an earlier version of the NPS \LaTeX\ template.
}
